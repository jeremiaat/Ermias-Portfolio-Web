# Jeremiah's Portfolio

A clean, responsive portfolio website showcasing my web development work. Built with modern web technologies and focused on user experience.

## Features

- Responsive design that works on all devices
- Dark mode with automatic system preference detection
- Projects loaded dynamically from JSON data
- Smooth animations and interactions
- Accessible and semantic HTML structure

## Technologies

- HTML5 for structure and accessibility
- CSS3 with modern layouts and animations
- JavaScript for interactivity and dynamic content
- No frameworks or build tools - just vanilla web technologies

## Project Structure

```
Portfolio Website/
├── index.html          # Main page structure
├── style.css           # All styling and themes
├── script.js           # Interactive features
├── projects.json       # Project data
└── README.md           # This file
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jeremiaat/Ermias-Portfolio-Web.git
   ```

2. Navigate to the project:
   ```bash
   cd Ermias-Portfolio-Web
   ```

3. Open `index.html` in your browser - no setup required

## Customization

### Adding Projects
Edit `projects.json` to add new projects:
```json
{
  "title": "Portfolio Website",
  "description": "clean and modern portfolio website",
  "stack": ["Html5", "css", "javascript"]
}
```

### Changing Colors
Modify CSS variables in `style.css` to customize the color scheme and theme.

## Browser Support

Works on modern browsers including Chrome, Firefox, Safari, and Edge.
